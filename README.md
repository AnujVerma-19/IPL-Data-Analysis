
# IPL Data Analysis Project

## Overview
This project analyzes IPL cricket data using Python and Power BI.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI

## Analysis Performed
- Top Winning Teams
- Toss Decision Analysis
- Top Run Scorers
- Top Wicket Takers

## Dashboard
Interactive Power BI dashboard with charts and filters.
